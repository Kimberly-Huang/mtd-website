# Meat the Data — Driving Omnichannel Conversion at JBS Swift

A single-page report website presenting our Capstone in Applied Analytics project: a data-driven strategy to lift JBS Swift's omnichannel penetration from a flat **2.94%** to the sponsor target of **6%**.

## View the site

Open `index.html` in any browser. The site is fully self-contained (one HTML file; charts load from a CDN).

## What's inside

- Executive summary, business challenge, and structural barriers
- Data foundation across 16 tables (25 months, 123M+ customer rows, 150M+ transactions)
- Exploratory findings with interactive charts (penetration trend, value gap, NPS by channel, product online share, seasonality, reachability)
- The Who / What / When / How / Where strategy framework
- The rules-based Priority Score model (P1 / P2 / P3 bands)
- Activation blueprint, delivery roadmap, and success criteria
